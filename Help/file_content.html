{file:content}

此参数返回当前选择文件的内容

1、仅在当前选择内容为"文本"文件时，此参数才有效
2、文本文件一般是 .txt .ini .ahk等文件

选项

[cp0] 编码设置

编码可以为下列值的其中一个:

  • UTF-8: Unicode UTF-8, 相当于 CP65001.
  • UTF-16: Unicode UTF-16, 小端字节顺序. 相当于 CP1200.
  • UTF-8-RAW 或 UTF-16-RAW: 像上面一样, 但创建新文件时不写入字节顺序标记.
  • CPnnn: 带数值标识符 nnn 的代码页. 请参阅 代码页标识符.
  • 空或省略: 系统默认的 ANSI 代码页, 这也是默认设置.

示例

假设当前选择的对象是文件:
C:\dir\menuz.txt
文件内容为:
E:\test.ini
菜单项的内容是:
notepad.exe "{file:content}"
会被解析为:
notepad.exe "E:\test.ini"
